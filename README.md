# Драгоценный проект

## Создать файл настроек

```shell
cp config.js.sample config.js
```

### Windows

```cmd
copy config.js.sample config.js
```

## Отредактировать настройки

Настройки находятся в `config.js`

## Установить зависимости и запустить

```
npm install
npm run start
```

### Или

```
yarn
yarn run start
```
